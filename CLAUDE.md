# Abount Next SQL

これは、SQLクエリを生成する関数を書き、特定のプログラミング言語用のモジュールを生成するためのDSLのプロジェクトです

注意事項
- 構文をアップデートするたびに、以下を保守してください
   - learn-next-sql スキルのプロンプト
   - LSPサーバー
   - 各バックエンド
- 参考として他のSQLやクエリのコードを与えた場合も、特定の業務要件を取り込まず、必ず一般化するようにしてください
- NextSQLのランタイムは絶対の特定のRDB実装やライブラリの実装に依存しないようにしてください。

対応バックエンド言語は現状、Rustのみです

VSCode拡張に関わる変更を行なった場合は必ずtagをpushして、新しいバージョンがリリースされるようにしてください

## Rustの書き方
- +#[allow(dead_code)] は絶対に使わないでください。もし、コードが未使用であれば、削除してください。
- +#[allow(unused_variables)] も同様に使わないでください。
